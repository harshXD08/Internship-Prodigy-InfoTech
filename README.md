# GPT-2 Text Generation

This project fine-tunes the GPT-2 language model using Hugging Face Transformers to generate motivational text.

## Technologies Used
Python
PyTorch
HuggingFace Transformers
Datasets

## Files
train.py – model training  
generate.py – text generation  
data.txt – training dataset  

## Installation

pip install -r requirements.txt

## Run Training

python train.py

## Generate Text

python generate.py
